# raccioly/homebrew-tap

Homebrew tap for [DocGuard](https://github.com/raccioly/docguard) — the enforcement tool for Canonical-Driven Development.

```bash
brew tap raccioly/tap
brew install docguard
docguard --version
```

The formula tracks the `docguard-cli` npm package; Homebrew installs Node.js 18+ as a dependency.
